# リニューアル — 式様変更ドキュメント

> 日時: 2026-06-10
> プロジェクト: 东方幻想麻雀 (Touhou Gensou Mahjong)

---

## 1. 開始画面 (Title Screen)

**変更前**: アプリ起動後、即座に牌局画面が表示されていた。

**変更後**: ゲーム開始前にタイトル画面が表示される。

- 背景：`assets/pic/desktop/` 内の4枚の画像からランダム選択、暗めのオーバーレイ付き
- タイトル：「東方幻想麻雀」＋英語サブタイトル（金色グラデーション）
- メニューボタン：
  - **单人模式** (シングルプレイヤー) — 選択可能。選人画面へ遷移
  - 组队模式、剧情模式、联机模式 — COMING SOON (disabled)

**新規ファイル**:
- `src/components/StartPage.tsx`
- `src/styles/title-screen.css` (背景・レイアウト・ボタンスタイル)

---

## 2. 選人画面 (Character Select)

**新規機能**: シングルプレイヤーモード選択後、キャラクター選択画面に遷移。

**レイアウト**:
- ヘッダー: 戻るボタン＋選択スロット (1P〜4P) のインジケーター
- 左サイドバー: チーム一覧 (th06〜th20、15チーム)
- 右グリッド: 選択したチームのキャラクター一覧
- フッター: 選択状況表示＋「开始游戏」ボタン (4人揃うまで無効)

**操作方法**:
- キャラクターをクリック → 空きスロットに割り当て
- 既に選択済みのキャラをクリック → 解除
- 上部スロットドットをクリック → そのスロットの選択を解除
- 4人選択後、「开始游戏」をクリック → 牌局画面へ

**新規ファイル**:
- `src/components/CharacterSelect.tsx`
- `public/characters.json` (108キャラクター、15チームのデータ)

## 3. キャラクターデータ JSON

`doc/abilities.md` をパースして `public/characters.json` を生成。
各エントリの構造:
```json
{
  "id": "th06-01",
  "nameCN": "露米娅",
  "nameJP": "ルーミア",
  "nameEN": "Rumia",
  "race": "妖怪（不明）",
  "ability": "操纵黑暗程度的能力",
  "icon": "th06-01-head",
  "normal": "th06-01-pic-01",
  "win": "th06-01-pic-02",
  "lose": "th06-01-pic-03",
  "skill": "th06-01-pic-04",
  "personality": "天真残虐，对自己的能力掌控不佳",
  "mahjong_skill": "Lv1(20) 暗黒 — 对手手牌模糊化 2 巡"
}
```

チーム構造:
```json
{
  "teamId": "th06",
  "teamName": "东方红魔郷",
  "members": [ ... ]
}
```

**新規ファイル**:
- `scripts/parse-characters.py` (パーサー)
- `public/characters.json` (生成データ)

## 4. ページルーティング

**変更前**: App.tsx が直接 GameTable をレンダリングしていた。

**変更後**: 3つのページ間を状態遷移:
| ページ | state | 説明 |
|--------|-------|------|
| Title | `'title'` | 開始画面 |
| Select | `'select'` | キャラ選択 |
| Game | `'game'` | 牌局 (GameTable) |

**変更ファイル**:
- `src/App.tsx` — 全面書き換え（ルーティング＋GamePage内部コンポーネント化）

---

## 5. カスタムキャラクター名対応

**変更前**: `TOUHOU_CHARACTERS` (types.ts) の固定4キャラが常に使用されていた。

**変更後**: `GameController` がコンストラクタで任意のキャラクター名配列を受け取れるようになり、
選人画面で選んだ4キャラの中国名が牌局に反映される。

**変更ファイル**:
- `src/game/GameController.ts` — コンストラクタ＋newGame/nextHand に characters パラメータ
- `src/game/gameEngine.ts` — `createInitialState(characters?)`
- `src/hooks/useGame.ts` — `useGame(characters?)` ＋ charsRef
